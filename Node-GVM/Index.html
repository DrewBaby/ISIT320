Introduction

Project Statement

The Game Club R US online application is an online membership tool that empower individuals with the tools needed to capture and track all past and current games within this. central tool. The tool has a vast data source providing by IGDB.COM providing rich information that gamers can enjoy such as extensive list of games, game platforms, past reviews, ratings, game artwork, character background stories, game cover, and much, much more.

Once enrolled, member will be able to build their own custom list of games based on their search preference. Once associated to their profile, members will be able to rate the games, review the rich content provided and also provide and exchange comments with other members about the games you are interested in.

As part of the roadmap, we plan on expanding sites to include streaming videos of game plays and the ability for members to upload custom covers and logos.

Solution

For our integrated development environment, we chose to use Visual Studio 2019 as the defacto tool. Main reason for this selection is as follow:

  • We will develop an ASP.NET Core 3.0 web based application and Microsoft Visual Studio 2019 comes fully ready to support this effort
  • Visual Studio 2019 is a full-featured integrated development environment
  • Visual Studio has all the features that a developer might need during his application development process
  • Azure Cloud integrations
  • Test Development Integrations
  • Easy to implement Debugging Tools and Break Points

Platform

For the platform, we chose to develop on Windows platform to support ease and simplifcation of the team to pickup and manage. Moving away from Microsoft Windows and Visual Studio 019 may have led to issues pertaining to learning curve for team members participating in the project. The KISS principle applies here for this project.

External Dependencies

To acheive our goal, we need the ability to leverage external services to provide the rich content to power bring life to the site.

IGDB

IGDB.com is a video game online data source providing interface for external consumers to access data of all games and details for each game stored in a central location

Through IGDB API, there are many endpoints used to return back specific data used supporting what is neede such as company name, game characters, expansion pack detail, game summary, online video, cover detail, game images, and much more.
For more details about the IGDB API, please visit the site here.

Google Identity

We wanted to add a visibly trusted and secure Sign In With Google button to your app. This will provide the ability to authenticate and identify members and tear down the anonymous behaviour of the internet.

Users sign-in to a Google Account once, without re-entering username or password when visiting your site. On return visits sign users in automatically or with one click.

Through the Google's OAuth 2.0 APIs, We were able to conform to the OpenID Connect specificatione used for authentication and authorization.

User Stories

The user stories listed below capture what the site should accomplish. The team and I reviewed and filter down a large list to the following to showcase the Game Club concept:

  1. As a user, I want to be able to sign up as a new Game Club R Us member.
  2. As a member, I want to associate multiple gamer handles, if needed, to my membership profile
  3. As a member, I would like to search through a pre-defined list of games published for every platform.
  4. As a member, I want to filter list of games by game genre
  5. As a member, I want to filter list of games by keyword to identify specific games I care about
  6. As a member, I want to filter a return list by platform to identify specific games I care about
  7. As a member, I want to filter a return list by genre to identify specific games I care about
  8. As a member, I want to list games I have selected assigned to my profile
  9. As a member, I would like to write comments about each of the games to share my opinion and to have a dialog with other members.

As a member, we strived to accomplish this task when delivering on the key features and functionalities of the web based members-only application.
